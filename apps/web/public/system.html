05

The laws

Non-negotiable
  • Vertical = probability. Halfway = a thin constant Press-Black seam at exactly 50%.
  • THE DRAW is a paper-band width (= pDraw) with zigzag teeth; the seam rides inside it.
  • Territory extent = exactly win probability; solid core → dot-dissolve at the working edge.
  • Market number ≠ crowd counts. Never blended.
  • One loud ground per surface; team colors only in the four slots.
  • Boxes-first grid; keyline ≈2%, cream border ≈5%; one diagonal max.
  • Type on bands/boxes, never floating. Doto for numbers, Anybody for screams.
  • Press texture only — halftone, grain, a hair of misregistration.
  • Every rail cell carries a job. Goals erupt only at the real goal mouth.
Banned — the diseases
  • True-hexagon soccer balls — only the 5-segment pinwheel pop-ball.
  • Speed-streaks, motion-blur — motion is stepped/snap/squash.
  • Glow, gradient mesh, drop-shadow blur, glassmorphism.
  • Sepia, scratches, torn/burnt paper, faux-vintage filters.
  • Sticker-bomb ornament — no decoration without a job.
  • Trophy / silverware in match frames.
  • Player faces or likenesses — the portrait is data.
  • Club crests, FIFA marks.
  • Two loud grounds touching; a probability in a display face.
  • A crowd count dressed as a percentage.
