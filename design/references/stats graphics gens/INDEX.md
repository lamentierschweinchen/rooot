# Stat-graphic generations — index

Owner drops raw ChatGPT exports (timestamped `.png`) into the **root** of this
folder. I sort them into the subfolders below with meaningful names, and note
which variant is CHOSEN. The chosen file is cleaned (transparency keyed /
recoloured as needed) and copied into `apps/web/public/plate/gens/` for live use.

Convention going forward: **raw drop → root → I sort here → clean copy → `plate/gens/`.**
